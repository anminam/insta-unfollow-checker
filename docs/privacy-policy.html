Privacy Policy

Last updated: March 4, 2026

Insta Unfollow Checker ("the Extension") is a Chrome browser extension that helps you identify Instagram users who don't follow you back. This privacy policy explains what data we collect, how it is used, and how it is stored.

1. Data We Collect

Data Type Purpose Source
Instagram following list Compare with followers to find non-mutual follows Instagram GraphQL API (using your existing web session)
Instagram follower list Compare with following to find non-mutual follows Instagram GraphQL API (using your existing web session)
Google email address Access control via authorized user list Google OAuth 2.0 (chrome.identity)

2. How Data Is Stored

All data is stored locally on your device. We do not operate any backend servers and do not store any of your data externally.

Storage Location Data Retention
localStorage (browser) Unfollow history, analysis snapshots, whitelist, user memos, preferences Until you clear browser data or uninstall
sessionStorage (browser) Analysis cache 10-minute TTL, cleared when tab closes
chrome.storage.local Auto-analysis snapshots, Google auth cache Auth cache: 24-hour TTL. Snapshots: until cleared

3. Third-Party Services

The Extension communicates with the following external services:

  • Instagram (instagram.com) — To fetch your following/follower lists and perform follow/unfollow actions. Uses your existing Instagram web session cookies. No additional login is required.
  • Google APIs (googleapis.com) — To authenticate your Google account and check the authorized user list via Google Sheets API. Only your email address is accessed.

We do not send data to any other third-party services, analytics platforms, or advertising networks.

4. Data Sharing

We do not sell, trade, or share your data with any third parties. Your Instagram data and Google email remain on your local device.

5. Permissions Explained

Permission Why It's Needed
cookies Read Instagram session cookies (csrftoken, ds_user_id) for API authentication
storage Store auto-analysis results and auth cache locally
alarms Schedule automatic background analysis (every 24 hours)
identity Google OAuth sign-in for access control
host_permissions (instagram.com) Access Instagram API endpoints for data fetching
host_permissions (cdninstagram.com, fbcdn.net) Load profile images from Instagram CDN
host_permissions (googleapis.com) Google Sheets API for authorized user list verification

6. Data Security

Since all data is stored locally in your browser, it is protected by your operating system and browser security. The Extension does not transmit your data to any server we operate.

7. Your Rights

  • View your data: All stored data is accessible in your browser's developer tools (Application tab).
  • Delete your data: Use the Extension's data management features (JSON backup/restore, CSV export) or clear your browser data.
  • Uninstall: Removing the Extension deletes all chrome.storage.local data. localStorage data can be cleared via browser settings.

8. Children's Privacy

This Extension is not intended for use by children under 13. We do not knowingly collect data from children.

9. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date.
