Instagram에서 나를 팔로우하지 않는 사람을 한눈에 찾아보세요.

🔍 주요 기능

• 팔로잉/팔로워 분석 — 전체 목록을 수집하여 맞팔 여부를 비교합니다
• 맞팔 안함 탐지 — 나만 팔로우하고 있는 계정을 즉시 확인
• 맞팔 목록 — 서로 팔로우하는 계정을 한눈에 확인
• 원클릭 언팔로우 — 개별, 선택, 전체 언팔로우 지원
• 예약 언팔로우 — 3~5분 간격으로 자동 처리하여 안전하게 언팔

📊 통계 & 추적

• 팔로워/팔로잉/맞팔안함 추이 차트
• 팔로워 변동 추적 (신규/이탈)
• 분석 이력 최대 20개 보관
• 스냅샷 비교로 변화 상세 확인
• 팔로워 비율 건강도 표시

🛡️ 데이터 관리

• 화이트리스트 — 보호할 계정을 지정하면 일괄 언팔에서 제외
• 유저 메모 & 태그 — 계정별 메모, 그룹 분류 (친구/셀럽/브랜드/업무)
• CSV 내보내기 — 분석 결과를 스프레드시트에서 확인
• JSON 백업/복원 — 전체 데이터를 안전하게 보관

⚙️ 편의 기능

• 다크모드 지원 (시스템 설정 자동 감지)
• 한국어/영어 지원
• 실시간 검색 & 필터
• 정렬 (이름순/인증우선/오래된순)
• 키보드 단축키 (Ctrl+A, Esc, Ctrl+F)
• Shift+Click 범위 선택
• 자동 분석 (24시간마다 백그라운드 실행)
• 뱃지 알림으로 변동 사항 즉시 확인

🔒 개인정보

• 모든 데이터는 브라우저 내 로컬에만 저장됩니다
• 별도의 서버로 데이터를 전송하지 않습니다
• Instagram 웹 세션을 그대로 활용 (추가 로그인 불필요)

📋 필요 권한

• instagram.com — 팔로잉/팔로워 데이터 조회
• googleapis.com — Google 로그인 접근 제어
• cookies — Instagram 세션 인증
• storage — 분석 결과 로컬 저장
• alarms — 자동 분석 스케줄링
